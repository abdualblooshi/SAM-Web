About SAM

SAM which is short for Smart Artificial Mirror, which is a mirror that uses Artificial Intelligence to assist it’s user with the daily tasks that he or she may have, SAM uses Facial Recognition and Speech Recognition Technology to perform its various functions, SAM uses an LED monitor that would show the Graphical User Interface.

The goal of this project was to build a working smart mirror that can reflect a face and show an interface at the same time, and it could recognize it’s user by using various functions and features of Artificial Intelligence, and developing this project would allow Smart Mirrors to evolve in the upcoming future.

Why did we choose this project?

It’s been years since Smart technology has been introduced to the world and it has become part of our daily routine, and there has been a rise of voice smart devices and smart phone personal assistant such as Siri and Google Assistant, we’re currently in a time where IoT devices help us in our daily lives, nowadays normal lights have become smart lights, even toasters are now smart, but mirrors still have not evolved which is why we as a group chose making a mirror which would be voice activated, uses facial recognition with the assist of Artificial Intelligence.

Hardware

SAM uses is a single board computer launched by ASUS in early 2017. Its physical size and GPIO pinout are designed to be compatible with the second-generation and later Raspberry Pi models. The first released board features 4K video, 2GB of onboard RAM, gigabit Ethernet and a Rockchip RK3288 processor running at 1.8 GHz.

Artificial Intelligence

Before initiating the work on SAM, we had to research more about Artificial Intelligence so that we would have the ability to create a Python Program which would use Artificial Intelligence, and during our research we have found out that Artificial Intelligence became an essential part of the technology industry.

Facial Recognition

In order for the AI to successfully recognize a person’s face, it must be trained a precise training using certain algorithms and python codes, the process of the training goes as following: Photos for example of Mohammed’s face are inserted into the training program (python software) → The AI tries to predict whether the picture given is a human face or not by using a Boolean Function → Face Recognized

Speech Recognition

Python is used for Speech Recognition, and Google API.

And the text below is an explaination of how it works
Device is listening... → User says something and the AI tries to recognize it → If it’s recognized the AI compares the words in the sentence to other words which were said while training the AI → matched sentences are defined as Entities if the device found a matching sentence it adds it as a parameter to a function as for example “User_Status”. → Google Speech API outputs Text To Speech.

Speech Synthesis

Many people have said that our project has the same voice as Google Assistant.

That is because SAM uses Google's Speech Synthesis inorder to speak back to it's users, when SAM talks back to it's user, the response is also shown on the user interface whether it's in figures or just plain text.
